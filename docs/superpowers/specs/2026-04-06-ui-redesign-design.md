# BIGSPACE UI Redesign Design Spec

## Overview

BIGSPACE GIS 웹 플랫폼의 전반적 UI 리디자인. 차분하고 깔끔한 스타일, 지도가 주인공이 되는 미니멀 GIS SaaS 방향.

**레퍼런스:** Mapbox Studio / Felt.com
**테마:** 라이트 기본 + 다크 모드 토글 지원
**핵심 전제:** 레이어 최대 150개 대응

## 2. 사이드바 (150개 레이어 대응)

### 2.1 구조

```
┌─────────────────────────── 280px ──┐
│ 로고 + BIGSPACE              [접기] │
├────────────────────────────────────┤
│ 검색바 (레이어 이름/타입 필터)       │
├────────────────────────────────────┤
│ ▼ 그룹명 (N)            [전체 토글] │
│   레이어 행 32px                    │
│   레이어 행 32px                    │
│ ▶ 그룹명 (N)            [전체 토글] │
│                                    │
│   ··· 가상 스크롤 (react-window) ···│
│                                    │
├────────────────────────────────────┤
│ 분석도구 (접기/펼치기)               │
├────────────────────────────────────┤
│ 사용자 정보              [설정][로그아웃] │
└────────────────────────────────────┘
```

### 2.2 레이어 행 (32px)

```
┌─ 3px 세로 컬러바
│ [드래그핸들] ● 레이어이름       478K  ◉
│              색상도트(12px)          가시성
└───────────────────────────────────────┘
```

- 호버: 배경 `--bg-surface`, 우측에 `⋯` 메뉴(스타일/삭제/속성)
- 활성(visible): `--text-primary`
- 비활성: `--text-tertiary`, 50% opacity
- 피처 수: `478K` 축약, `--text-tertiary`

### 2.3 그룹 헤더

- ▶/▼ 접기/펼치기
- 그룹명 + 레이어 수 `(N)`
- 우측: 전체 ON/OFF 토글
- 접힌 상태에서는 그룹명과 수만 표시

### 2.4 검색

- 상단 고정, 사이드바 내부
- 레이어 이름, 타입(WMS/WFS/Vector) 필터
- 결과 없음: "일치하는 레이어가 없습니다"

### 2.5 드래그 정렬

- 그룹 내 레이어 순서 변경 (zIndex)
- 드래그 핸들(⠿) 호버 시 표시

### 2.6 사이드바 접기

- 48px 아이콘 모드
- 로고 아이콘만 표시
- 레이어 아이콘 + 가시성 도트만

## 3. 지도 컨트롤

### 3.1 검색바

- `max-width: 400px`, 상단 중앙
- 라운드(`--radius-md`), 박스 섀도우
- 배경: `--bg-primary` + 80% opacity backdrop-blur
- placeholder: "주소, 지명 또는 좌표 검색..."

### 3.2 줌 컨트롤

- 우측 하단, 세로 배치
- `+` / 줌레벨(`Z14`) / `-` 하나의 컨트롤로 통합
- 줌레벨: `--font-mono`, `--text-xs`

### 3.3 도구 버튼

- 우측 상단, 세로 배치
- 각 버튼에 툴팁
- 활성 상태: `--accent` 배경 하이라이트

### 3.4 좌표 표시

- 좌측 하단
- `--font-mono`, `--text-xs`
- `37.55°N, 126.97°E  |  1:250,000`

### 3.5 베이스맵 피커

- 하단 중앙, 드롭업 방식
- 썸네일 프리뷰 (64x64) + 이름
- 3~4개 옵션: VWorld 기본, VWorld 위성, OSM, Naver

---

## 4. 로그인 페이지

- 가운데 정렬 카드 (`max-width: 380px`)
- 배경: 은은한 지형도 패턴 또는 `--bg-secondary` 그라디언트
- 로고 + "BIGSPACE" + 부제 "공간정보 통합 플랫폼"
- 이메일/비밀번호 입력
- 로그인 버튼 (`--accent`)
- Google OAuth 버튼
- 하단: 회원가입 | 비밀번호 찾기 링크

---

## 5. 상태/피드백

| 상태 | 처리 |
|------|------|
| 레이어 로딩 | 레이어 행에 스켈레톤 애니메이션 |
| WFS/MVT 로딩 | 지도 위 작은 스피너 + "불러오는 중..." |
| 검색 결과 0개 | "일치하는 레이어가 없습니다" |
| 에러 | 토스트 알림 (하단 우측, 3초 자동 닫힘) |
| 줌 부족 안내 | 해당 레이어 행에 "확대하면 표시됩니다" |

---

## 6. 다크/라이트 전환

- 사이드바 하단 설정 영역에 테마 토글
- CSS 변수 기반 전환 (`.dark` 클래스)
- 전환 시 150ms transition
- 지도 타일은 테마 영향 없음 (베이스맵 자체 스타일)

---

## NOT in scope (이번에 안 함)

- 레이어 0개 빈 상태 UI
- 모바일/태블릿 반응형 (데스크톱 우선)
- 설정 페이지 리디자인
- 분석도구 패널 상세 디자인
- 접근성(a11y) 전면 적용
- 애니메이션/모션 디자인
