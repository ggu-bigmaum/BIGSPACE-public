## 부록 - 결함 등급 기준

| 등급 | 정의 | 수정 기한 |
|------|------|----------|
| Critical | 시스템 정지, 데이터 손실, 보안 파열 | 5일 이내 |
| High | 주요 GIS 기능 미구현 또는 결과 산출 불가 | 4일 이내 |
| Medium | 기능은 동작하나 결과값 오차 있음 | 3일 이내 |
| Low | 오타, UI 배치, 경미한 입력 제한 미비 | 2일 이내 |
