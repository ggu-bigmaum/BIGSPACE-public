## 2. 데이터 관리

| # | 항목 | 상태 | 비고 |
|---|------|------|------|
| 2-1 | 레이어별 데이터 출처·기준연도·갱신주기 표시 | ❌ | 공공기관 납품 시 필수 메타데이터. 심사관이 바로 물어보는 항목 |
| 2-2 | 다양한 좌표계 입력 지원 | ⚠️ | EPSG:5181→4326만 지원. 공공데이터는 5174·5179 등 다양 |
